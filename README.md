# E-Commerce Sales & Customer Insights Dashboard

## Project Overview
This project is a comprehensive Data Analytics solution designed to extract, process, and visualize e-commerce business performance. It demonstrates skills in **Python** for data engineering, **SQL** for complex business analysis, and **Interactive Dashboards** for storytelling.

## Key Features
- **Sales Performance Tracking**: Monthly trends, revenue growth, and profit margin analysis.
- **Customer Segmentation**: Identification of top customers and repeat purchase behavior.
- **Product Profitability**: Category-wise performance and best-selling product identification.
- **Regional Insights**: Geographic sales distribution and shipping efficiency.
- **SQL Analysis**: Business-critical queries for churn, LTV, and cohort analysis.

## Tools Used
- **Python (Pandas, NumPy)**: Data cleaning and feature engineering.
- **SQL**: Data extraction and business logic implementation.
- **Live Dashboard**: Interactive web-based analytics (React + Recharts).
- **Power BI (Conceptual)**: Designed following professional BI UI/UX standards.

## Project Structure
- `/data`: Source e-commerce dataset.
- `/scripts/preprocessing.py`: Python cleaning script.
- `/scripts/analysis.sql`: SQL business query library.
- `/src`: Dashboard application source code.

## Business Insights
1. **Peak Season**: November and December show a 40% spike in electronics sales.
2. **Customer Retention**: 25% of customers are responsible for 60% of total revenue.
3. **Logistics**: Standard shipping in the West region has the highest delay rate (5.2%).
